## Quick start

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/hetari/shadcn-docs-template.git docs
    cd docs
    ```
  </Step>
  <Step title="Install dependencies">
    ```bash
    bun install
    ```
  </Step>
  <Step title="Start the development server">
    ```bash
    bun dev
    ```
    Open [http://localhost:3000](http://localhost:3000) to see your docs site.
  </Step>
  <Step title="Customize your site">
    Update `app/lib/siteConfig.ts` with your project name, URL, and GitHub repo. Then add content to `content/docs/`.
  </Step>
</Steps>
